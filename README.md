## üöÄ Installation

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- Web server (optional, for local development)

### Quick Start

1. **Clone the repository**
   \`\`\`bash
   git clone https://github.com/Elly-L/portfolio.git
   cd portfolio
   \`\`\`

2. **Open in browser**
   \`\`\`bash
   # Option 1: Direct file opening
   open index.html
   
   # Option 2: Local server (recommended)
   python -m http.server 8000
   # Then visit http://localhost:8000
   \`\`\`

3. **Customize content**
   - Update personal information in `index.html`
   - Modify colors in CSS custom properties
   - Replace project images and links
   - Update CV file path

## üíª Usage

### Customization Guide

#### 1. Personal Information
Update the following sections in `index.html`:
- Hero section name and title
- About section biography
- Skills categories and tags
- Project information and links
- Contact form email address

#### 2. Color Scheme
Modify CSS custom properties in `styles.css`:
\`\`\`css
:root {
  --cosmic-purple: #6366f1;
  --electric-blue: #3b82f6;
  --neon-green: #10b981;
  /* Add your custom colors */
}
\`\`\`

#### 3. Content Updates
- **Projects**: Update project cards with your own work
- **Skills**: Modify skill categories and technologies
- **Images**: Replace placeholder images with your own
- **Links**: Update all external links to your profiles

#### 4. CV Integration
- Place your CV PDF in the `cv/` directory
- Update the file path in both top action buttons and contact section

## üì± Responsive Design

### Breakpoint Strategy
\`\`\`css
/* Mobile First Approach */
/* Base styles: 320px+ */

@media (max-width: 480px)  { /* Extra small phones */ }
@media (max-width: 768px)  { /* Standard mobile */ }
@media (max-width: 992px)  { /* Tablets */ }
@media (min-width: 1200px) { /* Large desktops */ }
\`\`\`

### Mobile Optimizations
- **Navigation**: Icon-only mobile menu
- **Typography**: Scaled font sizes
- **Layout**: Single-column layouts
- **Touch Targets**: Minimum 44px tap areas
- **Performance**: Optimized animations
- **Content**: Prioritized information hierarchy

### Key Responsive Features
- **Flexible Grid Systems**: CSS Grid and Flexbox
- **Fluid Images**: Responsive image scaling
- **Adaptive Navigation**: Context-aware menu
- **Touch-Friendly**: Optimized for mobile interaction
